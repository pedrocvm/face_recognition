### :dvd: **Running**
```bash
# Clone this repository
$ git clone https://github.com/pedrocvm/face_recognition

# Access the project folder in terminal/cmd
$ cd face_recognition

# Running with Docker
$ docker-compose up -d

or locally

# Install the Dependencies
$ yarn

# Run the Back-End App
$ yarn start:dev

# Run the Front-End App
$ yarn serve
```
